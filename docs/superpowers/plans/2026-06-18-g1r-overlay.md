# G1R-Overlay Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Aus dem G1R-Diagnose-Widget drei theme-fähige OBS-Overlays (Livebar, News-Ticker, Career-Card) + ein angereichertes Daten-Backend machen.

**Architecture:** Lokaler UE4SS-Mod schreibt `g1r-state.json` (+ neu `g1r-totals.json` für persistente Gesamtwerte/Rekorde). `server.py` (localhost:9210) reichert `/state` mit „stärkste Waffe" und „stärkster nutzbarer Zauber" aus zwei Wiki-Mappings an und liefert UI-Labels per `?lang`. Drei statische Widgets pollen `/state`.

**Tech Stack:** Lua 5.4 (UE4SS), Python 3 (stdlib http.server), Vanilla HTML/CSS/JS. Theme via `widgets/_theme.css` + server-injiziertes `data-theme`.

**Test-Realität:** server.py-Logik → pytest (echtes TDD). Lua → `luaparser`-Syntaxcheck + in-game. Widgets → Browser/`/state`. Spec: `docs/superpowers/specs/2026-06-18-g1r-overlay-design.md`.

# in server.py, bei den _load_*-Helfern
def _load_json_map(path):
    try:
        with open(path, "r", encoding="utf-8") as fh:
            return {k: v for k, v in json.load(fh).items() if not k.startswith("_")}
    except Exception:
        return {}

WEAPON_DAMAGE = _load_json_map(os.path.join(os.path.dirname(os.path.abspath(__file__)), "weapon_damage.json"))

def strongest_weapon(items):
    """Klassenname der staerksten Waffe im Inventar (hoechster Wiki-Schaden) oder None."""
    best, best_dmg = None, -1
    for it in (items or []):
        dmg = WEAPON_DAMAGE.get(it.get("name"))
        if dmg is not None and dmg > best_dmg:
            best, best_dmg = it.get("name"), dmg
    return best
```

SPELL_CIRCLE = _load_json_map(os.path.join(os.path.dirname(os.path.abspath(__file__)), "spell_circle.json"))

def strongest_usable_spell(items, magic_circle):
    """Nutzbare Rune (Kreis <= magicCircle) mit dem hoechsten Kreis. None wenn keine nutzbar."""
    try:
        mc = int(magic_circle)
    except (TypeError, ValueError):
        return None
    best, best_circle = None, 0
    for it in (items or []):
        circ = SPELL_CIRCLE.get(it.get("name"))
        if circ is not None and circ <= mc and circ > best_circle:
            best, best_circle = it.get("name"), circ
    return best
```

## Etappe 3 — Mod: Persistenz, Rekorde, Damage-Hook

### Task 5: Persistente Gesamtwerte (g1r-totals.json)

**Files:**
- Modify: `g1r-local/G1RExport/scripts/main.lua`

- [ ] **Step 1: State + Lade-Funktion**

Bei den State-Vars (oben):
```lua
local TOTALS_PATH = [[C:\obs-g1r\g1r-totals.json]]
local totals = { damageTaken = 0, healthRegen = 0, manaSpent = 0, manaRegen = 0, xpGained = 0, distanceM = 0, steps = 0,
                 recHardestTaken = 0, recManaTick = 0, recRunM = 0,
                 recHardestDealt = 0, damageDealt = 0 }
-- Beim Start laden (sehr simpler Parser: Zahlen je Key).
pcall(function()
    local f = io.open(TOTALS_PATH, "r"); if not f then return end
    local s = f:read("*a"); f:close()
    for k in pairs(totals) do
        local n = s:match('"'..k..'"%s*:%s*(-?%d+%.?%d*)')
        if n then totals[k] = tonumber(n) end
    end
end)
local lastTotalsWrite = 0
```

- [ ] **Step 2: In tick die Session-Deltas auch auf totals addieren**

Direkt nach der Session-Delta-Berechnung (`sess.*`): die jeweiligen Deltas zusätzlich auf `totals.*` addieren (gleiche Logik, nur kumulativ). Strecke: `totals.distanceM` aus `totalDistCm` neu setzen ist falsch (Session) → stattdessen das horizontale Delta `d/100` auch auf `totals.distanceM` addieren, `totals.steps` analog.

- [ ] **Step 3: totals periodisch schreiben (alle ~10 s, nicht jeden Tick)**

```lua
-- am Ende von tick(), nach dem state-Write:
lastTotalsWrite = lastTotalsWrite + 1
if lastTotalsWrite >= 40 then  -- ~10 s bei 250 ms
    lastTotalsWrite = 0
    local parts = {}
    for k, v in pairs(totals) do parts[#parts+1] = string.format('"%s":%.2f', k, v) end
    local tf = io.open(TOTALS_PATH, "w")
    if tf then tf:write("{" .. table.concat(parts, ",") .. "}"); tf:close() end
end
```
