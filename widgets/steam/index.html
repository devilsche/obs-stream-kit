Steam Widgets — overview

All Steam-driven browser sources at a glance. Click a tile to open the widget URL in a new tab. Each card shows the recommended OBS browser-source size and the URL parameters you can pass — every widget also has its built-in filter panel that writes the same parameters back to the URL.

Widgets

now-playing.html

open
Bottom-left card with avatar, persona, current game, playtime, achievement progress and live player count.
520×180 · bottom-left anchor
  • pollMsnow-playing refresh rate (default 10000)
  • livePlayers0/1 — show live player count (default 1)
  • playersPollMsplayer count refresh (default 60000)
  • scaleglobal zoom factor
  • filter0 hides the live filter bar

popup.html

open
Combined slide-in popup — alternates between now-playing ribbon and achievement unlocks. Achievements interrupt, gold glow on rare unlocks.
540×140 · center anchor
  • nowPollMsnow-playing poll (default 600000 = 10 min)
  • achPollMsachievement poll (default 5000)
  • durationMsshow duration (default 8000)
  • gapMsgap between popups (default 1500)
  • rarePctrare threshold % (default 5)

achievement-popup.html

open
Pop-up just for achievement unlocks. Separate from the combined popup — uses its own deduplication key so both can run side by side. Rare unlocks (≤ rarePct %) glow gold.
540×140 · center anchor
  • durationshow duration in seconds (default 8)
  • gapgap between popups in seconds (default 1.5)
  • pollMspoll interval (default 5000)
  • rarePctrare threshold % (default 5)

achievement-browser.html

open
Tool-Page (nicht für OBS): Übersicht ALLER in der DB gespeicherten Achievements. Filter pro Game, Suche, Sort nach Datum/Rarität. Klick triggert Replay — Popup-Widget feuert das gewählte Achievement neu mit Sound. Zeigt Unlock-Datum + globalPct pro Eintrag.
Tool · im Browser-Fenster
  • —kein URL-Param, alles via UI

achievement-feed.html

open
Always-on rotating feed of the latest N unlocks across all games. Persistent — independent of the popup queue. Rare unlocks get a gold badge.
520×120 · anywhere
  • limitpool size (default 20)
  • rotateMsper-entry rotation (default 8000)
  • refreshMsre-fetch interval (default 60000)
  • rarePctrare threshold % (default 5)
  • headeroverride heading text

games-ticker.html

open
Library rotator: filterable by All / Co-op / Multiplayer with configurable sort, pool size, recency window. Reads from the local cache — survives Steam delistings (e.g. UT2004).
520×120 · anywhere
  • kindall / coop / multiplayer (default all)
  • sortplaytime / recent / random / name
  • playedSinceDaysonly games touched in last N days (0=off)
  • rotateMsper-entry rotation (default 8000)
  • cyclePauseMspause after last item (default 0)
  • minPlaytimeminimum total minutes (default 0)
  • limitpool size (default 30)
  • headerTitleoverride heading text

games-ticker.html · Wanna play?

open
Preset for chat engagement: random pick from multiplayer games that you've touched in the last 2 years. Header says Wanna play? — chat sees something they could join.
520×120 · anywhere
  • kindmultiplayer
  • sortrandom
  • playedSinceDays730 (≈ 2 years)
  • cyclePauseMs10000 (pause between cycles)
