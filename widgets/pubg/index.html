PUBG Widgets — overview

All PUBG widgets grouped by stream scene. Click a tile to open the widget URL in a new tab. Each tile shows the recommended OBS browser-source size and the key URL parameters — every widget has its built-in filter panel that writes the same parameters back to the URL. Most widgets hide themselves when no recent match is cached (override via ?ignoreStale=1).
